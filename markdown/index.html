markdown常用语法解释和说明

演示Markdown的基本语法, 点击Raw 可以看源代码，更好的掌握。

大标题

大标题，类似html的\<h1>
你只要在标题下面一行紧跟输入=====即可

大标题

大标题也可以使用 # 空格后，输入内容，来表示，类似html的\<h1>

中标题

中标题，类似html的\<h2>
你只要在标题下面一行紧跟输入------即可

中标题

中标题也可以使用 ## 空格后，输入内容来表示，类似html的\<h2>

小标题

小标题的格式如下 ### 空格后，输入内容，来表示，类似html的\<h3>

单行文本框

这是一个单行的文本框,只要1个Tab或者4个空格再输入文字即可

多行文本框

这是一个有多行的文本框
你可以写入代码等,每行文字只要输入1个Tab或者4个空格再输入文字即可

在多行文本框里输入一段代码,来一个PHP版本的HelloWorld吧

public function helloword() {
    echo 'Hello word';
}

加强的代码块,定格用``` 开头和 ```结尾，中间写代码。有行数和高亮提示

     echo 333;

链接

用[文字说明] (链接\ "Title") 来表示：

1.点击这里你可以链接到www.google.com
2.点击这里我你可以链接到我的主页，鼠标放上去有title

只是显示图片

用![] (图片地址 "图片title说明") 来表示：

想点击某个图片进入一个网页,比如我想点击github的icorn然后再进入www.github.com

区块引用

文字被些字符包围

只要在文字前面加上>空格即可

如果你要换行的话,新起一行,输入>空格即可,后面不接文字

或者这一行后面加\<br>,下一行也就能自动换行了 但> 只能放在行首才有效

区块多重包围引用

文字被些字符包围开始

只要再文字前面加上>空格即可

如果你要换行的话,新起一行,输入>空格即可,后面不接文字

但> 只能放在行首才有效

特殊字符处理

有一些特殊字符如<,#等,只要在特殊字符前面加上转义字符\即可
你想换行的话其实可以直接用html标签\<br />

无序列表

用 * 或者 - 或者+ 空格后，输入内容 ，即可显示无序列表，类似html的\<ul>

  • 在行首加* ，空格后输入内容即可
  • 或者在行首加-， 空格后输入内容即可
  • 或者在行首处加+，空格后输入内容即可

有序列表

用数字(1,2,3,4.....)加. 空格后，输入内容即可， 类似于似html的\<ol>

  1. 这是1
  2. 这是2
  3. 这是8，但是会显示3

分块显示

2块之间，输入一长串的----即可。记住：----上下都要有空格，不然就变成了大标题了。

这是一块

这是另一块

小代码块框框标识

用`const php ` 表示即可，显示很漂亮很显著

比如我们用git init来初始化git 版本库

粗体字

用** 文字 ** 来标识一个粗体字，类似html的\<strong>\</strong>
这是粗体

斜体

用* 文字 * 来标识一个斜体：这是斜体

删除线

用\~\~ 文字 \~\~ 来标识一个带有删除线的字体：~~有删除~~

表格

就可以绘制出一个下面的表格了；

项目 价格 数量
计算机 $1600 5
手机 $12 12
管线 $1 234
